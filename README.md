# HR Analytics Dashboard (Excel Project)

**Objective:** Analyze employee data to extract workforce insights, attrition trends, and salary patterns for an Indian company.

**Files included:**
- `Data/hr_raw_data.xlsx` - Raw employee dataset (EmpID, Name, Gender, Department, Location, HireDate, ExperienceYears, Salary, Status)
- `Reports/hr_analysis_report.xlsx` - Summary sheets (Dept_Summary, Gender_Summary, Insights)
- `Visuals/hr_dashboard.png` - Dashboard image (employees by department, attrition rates, salary distribution)

**Techniques & Functions used:**
- Data cleaning and aggregation
- Excel formulas: AVERAGE, COUNTIF, SUMIF, DATEDIF (tenure)
- Department-level summaries and attrition calculations
- Charts for visualization and storytelling
